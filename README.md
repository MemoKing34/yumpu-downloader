# Yumpu Downloader

This is a simple program that downloads pdf files from yumpu. 

## How to use it

Simply clone this repository, install dependencies and run it.

```bash
git clone https://github.com/MemoKing34/yumpu-downloader.git
cd yumpu-downloader
pip install -r requirements.txt
python3 main.py
```

It will download files current directory. Right now you can't choose a different output directory.
